Designing Sounds - A software engineering group project done in Autumn Term of 3rd year Computing at Imperial College London. With guidance from our group supervisor Mark van der Wilk we built a MacOS application where users can draw waveforms and play with them in real time to make interesting sounds and noises. We had to develop new and faster sound interpolation techniques in order to be able to do the modelling in real time. The application can take any either direct user input or use a MIDI controller to connect instruments as well as sound application such as Logic Pro.
